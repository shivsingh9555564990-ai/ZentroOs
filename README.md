# ⚡ ZentroOs — AI Growth OS for Creators
Premium AI SaaS for YouTube & Instagram creators.

- 🤖 AI Title Generator, Description Writer, Tag Generator
- 📸 AI Caption Generator, Hashtag Generator, Bio Optimizer
- 🔐 Firebase Auth (Email + Google)
- 🧠 OpenRouter AI with real-time free model selection
- 🎨 Premium glassmorphism dark UI

## Stack
React + Vite + TypeScript + Tailwind CSS + Firebase + OpenRouter AI
